Description

For this project, I made a VR Game in Unity using the Oculus development library. The game is a booth-style shooting gallery inspired by classic carnival games. It featured custom interaction mechanics, custom 3D models, fully 3D interaction system, graphics, and sound, and custom game logic. The grabbing, UI interactions, and movement systems were custom created for this game. The gun, targets, and table were all created by hand in Blender.
