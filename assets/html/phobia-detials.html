Project information

  • Demo Video: https://youtu.be/Xlnpdx1EJ1E

Description

For this group project, We created an application that helps people face their phobias in a simulated environment in VR. While this may seem like it would not be effective, one of the most powerful aspects of VR is its ability to trick our brains into thinking that what we see is what is really happening. This program was made in a way that takes advantage of that and leverages VR’s capability to help patients deal with phobias by exposing them to it and making them experience it without actually experiencing it. Currently, we have two rooms: A Public speaking phobia and height phobia room. They both attempt to create an environment where the phobia can easily be stimulated so that the patient can deal with their fear. This program was designed in a way that allows for many more “rooms” to be added in the future. This program also has a way to record patient data such as head and hand movements to a local file to be analyzed later.
